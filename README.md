# Azul

## How to run

1. Create a virtual environment:

    Linux:

    ```
    python3 -m venv venv
    ```


2. Activate the virtual environment:

    Linux:

    ```
    source venv/bin/activate
    ```

3. Install the required packages:

    Linux:

    ```
    pip install -r requirements.txt
    ```

4. Run the application:

    Linux:

    ```
    python3 main.py
    ```
